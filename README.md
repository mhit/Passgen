# Passgen - 日本語読みパスワード生成器

読める・覚えられる・安全なパスワードを生成する静的Webツール。

日本語の名前や単語をローマ字で組み合わせ、特殊文字・数字を挿入することで、**読み方がわかるパスワード**を生成します。電話口やメモでの伝達が容易です。

## 生成例

| パスワード | 読み方（カタカナ） | 読み方（ローマ字） |
|-----------|-------------------|-------------------|
| `!Idashin342` | イダ-シン | ida-shin |
| `?KazuhoHiwa7` | カズホ-ヒワ | kazuho-hiwa |
| `Ayaka3@Jio` | アヤカ-ジオ | ayaka-jio |
| `9KaaDaiki$` | カア-ダイキ | kaa-daiki |

## 機能

### パスワード生成
- 3,200語超の辞書（日本語の名前・一般語・造語）から2語を組み合わせ
- 10種の配置パターン（記号・数字の挿入位置）をランダム選択
- パスワード長: 短め(8-10) / 標準(10-14) / 長め(14-18)
- 特殊文字セット: 標準 / 安全 / 全て
- 大文字・数字・特殊文字の有無を個別設定可能

### 2つの運用モード

**単体モード**
- 候補パスワードを複数生成し、好きなものを選択
- クリックでクリップボードにコピー

**一括発行モード**
- ユーザーリスト（名前, ログインID）をテキスト入力またはCSVファイル読込
- 全ユーザーに対して一括でパスワードを生成
- 個別に再生成も可能

### 出力

**アカウント通知書（A4印刷/PDF）**
- ユーザーごとに1ページのA4通知書を生成
- 組織名・システム名・ログインID・URL・初期パスワード・読み方を記載
- 受領者署名欄・注意事項つき
- ブラウザの「PDFに保存」でPDF出力可能

**一覧印刷**
- 全パスワードをテーブル形式で一覧印刷

**CSV出力**
- 名前・ID・パスワード・読み方をCSV出力（BOM付きUTF-8）

## 使い方

1. `index.html` をブラウザで開く（サーバー不要）
2. パスワード設定を調整して「生成」ボタンを押す
3. 通知書情報を入力して「アカウント通知書」で印刷/PDF保存

## ファイル構成

```
Passgen/
├── index.html   ... UI + ロジック
├── dict.js      ... 辞書データ（3,200語超）
└── README.md
```

## 技術仕様

- **サーバレス**: 静的HTML + JavaScript のみ、サーバー不要
- **フォント**: Google Fonts（Noto Sans JP）を使用（印刷時も適用）
- **辞書ソース**: [gimei](https://github.com/willnet/gimei) の日本語名前データ + 一般語彙
- **印刷方式**: `window.print()` ベース（ブラウザの印刷ダイアログ経由でPDF保存可能）
- **組み合わせ数**: 3,200語 x 3,200語 x 10パターン x 記号 x 数字 = 数千万通り以上
